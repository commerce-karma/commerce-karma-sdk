# Installation

Install the sdk via npm.

## Use a package manager (recomended)

### npm

```sh
npm install commerce-karma-sdk
```

### yarn

```sh
yarn add commerce-karma-sdk
```

### pnpm

```sh
pnpm add commerce-karma-sdk
```

## bun

```sh
bun add commerce-karma-sdk
```

# Build from source

Make sure you have Node.js and pnpm installed first.

1. Clone the repository

```sh
git clone https://github.com/commerce-karma/commerce-karma-sdk
```

2. Run the build

```sh
pnpm run build
```

3. Import the sdk in your project from `/commerce-karma-sdk/dist`
